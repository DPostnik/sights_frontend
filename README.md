# Объекты Исторической Значимости

#### Для запуска клиентской системы:

<ol>
  <li>npm i</li>
  <li>npm run start</li>
</ol>

### Использованные технологии:

Node version: 14.15.5

<ul>
  <li>Angular2+</li>
  <li>RxJS</li>
  <li>NGXS</li>
  <li>MapBoxGL</li>
</ul>
